# Issue Packets

Store one Markdown packet per delivery issue in this folder.

## Naming

- Preferred: `<ISSUE-ID>-<short-slug>.md`
- Example: `AGE-272-workflow-docs-alignment.md`

## Packet Requirements

Each packet should include, at minimum:

- Metadata
- Scope budget
- Split triggers
- Objective
- Scope and non-goals
- Constraints
- Acceptance criteria
- Verification commands
- Stop/split rule
- Definition of done

Use `CB-001-codex-desktop-hook-discovery.md` as a reference packet.
